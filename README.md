# External-Raw-Leads-Flow
Using Google Apps Script (GAS) to implement automatic data transfer across different sources/platforms.

> ## Project Overview
> 
> 
> This part of the project handles the automated flow of lead data.
> ### Completed Workflows
> 
> 
> * **WP Contact Form 7 → Raw GSheet:** Captures external leads from the WordPress site.
> * **Raw GSheet → CRM GSheet:** Cleans and formats data for the main CRM.
> 
> 
> ### Historical/Alternative Flows
> 
> 
> * **GForm → CRM GSheet:** Initial prototype developed to test data mapping. *Note: Client later confirmed this was not required for the final workflow.*
>
